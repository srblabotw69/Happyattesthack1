# Happyattesthack1
 Welcome to Happy Attest! This is a one stop shop for creating Ethereum attestations via EAS (Ethereum Attestation Service).  
 
# installation

1.  cd happyattest  

npm install  


2.  cd vc-server  

npm install  

# config

Ensure to change appropriate config values:  

1.  happyattest/.env
2.  vc-server/.env

# run

1. cd happyattest  

npm start  

2. open a second terminal  

cd vc-server  

node src
